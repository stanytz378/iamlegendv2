  ### ✨ What makes IAMLEGEND different?
  
  | 🚀 Feature | 💡 Description |
  |---|---|
  | **Zero‑config plugins** | Drop a `.js` file in `plugins/` – auto‑loaded |
  | **5 database backends** | MongoDB, PostgreSQL, MySQL, SQLite, or JSON |
  | **Smart group protection** | Anti‑spam, anti‑link, anti‑badword, anti‑tag abuse |
  | **Role hierarchy** | Owner, Sudo, Admin, User – fine‑grained permissions |
  | **Scheduled messages** | Natural language timers (e.g. `in 2 hours`) |
  | **AI Chatbot** | Per‑group toggleable AI conversations |
  | **Privacy control** | Full WhatsApp privacy management via commands |
  | **Live polls** | Create polls with real‑time vote tracking |
  | **Broadcast engine** | Bulk message groups or DMs at once |
  | **Auto‑reply with variables** | Supports `{name}`, `{group}`, etc. |
  | **Games** | TicTacToe and more built in |
  | **Multi‑platform** | Termux, VPS, Heroku, Render, Railway, Koyeb, Fly.io, Replit, Docker |

  ## ✨ Features (Detailed)
  
  | Area | Features |
  |---|---|
  | **Core** | 250+ commands, auto-loading plugins, multi‑database, event system |
  | **Moderation** | Kick, ban, warn, promote, demote, mute, anti‑spam, anti‑link, anti‑badword |
  | **Group Tools** | Welcome/goodbye messages, polls, tag all, group info, invite |
  | **Fun** | Sticker maker, image effects, memes, games (ttt, trivia) |
  | **AI & Utilities** | Chatbot (simsimi/llama), translate, weather, news, lyrics, QR code |
  | **Privacy** | Read receipts, online status, typing indicator, call rejection, block/unblock |
  | **Media** | YouTube audio/video download, Instagram, TikTok, Twitter/X, Facebook |
  | **Database** | MongoDB, PostgreSQL, MySQL, SQLite, JSON files (auto‑switch) |

  ## 📌 Requirements
  
  | Requirement | Minimum |
  |---|---|
  | Node.js | 20.x or higher |
  | npm | 8.x or higher |
  | Git | any recent |
  | ffmpeg | latest (for media) |
  | libvips | latest (image processing) |
  | libwebp | latest (stickers) |

  ## ⚡ Quick Start
  
  ```bash
  git clone https://github.com/Stanytz378/IAMLEGEND.git
  cd IAMLEGEND
  npm install
  cp sample.env .env
  # Edit .env – add SESSION_ID and OWNER_NUMBER
  npm start
